# AirPodsClient
AirPods Client for Windows

| Color | Detail                                                  |
| ------------------- | ------------------------------------------- |
| White      | Scanning Devices      |
| Orange           | AirPods Pro    |
| Light Blue              | AirPods 2              |
| Red           | AirPods Unknown   |
* Since I can't test AirPods 1, the icon will be marked red.

## Requirements
- Python 3
- `pip install bleak pybluez infi.systray`
