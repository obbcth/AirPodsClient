# AirPodsClient
AirPods Client for Windows

## Requirements
- Python 3
- `bleak infi.systray (modified)`

## Usage
1. Clone this project.
1. Install Python 3 and run `pip install bleak pybluez infi.systray`.
1. Run `pythonw.exe AirPodsClient.py`.
